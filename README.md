# 🎌 Naruto vs Sasuke AR Hand Tracking System

A real-time AR web application that brings anime-style effects to life using hand gestures and computer vision.

---

## 🚀 Project Overview

This project was built as a fun AR experience for anime fans. It uses webcam-based hand tracking to trigger Naruto and Sasuke effects in real time.

---

## ✨ Features

- 🖐️ Real-time hand tracking using webcam  
- 🌀 Right hand → Sasuke Chidori effect  
- ⚡ Left hand → Naruto Rasengan effect  
- 📈 Dynamic power scaling based on finger movement  
- 🎥 Smooth AR video overlay effects  
- 🎮 Interactive gesture-based system  

---

## 🛠 Tech Stack

- HTML, CSS, JavaScript  
- MediaPipe Hands  
- Computer Vision concepts  
- WebRTC (webcam integration)
